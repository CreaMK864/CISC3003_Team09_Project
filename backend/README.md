# Chatbot Backend

A FastAPI-based backend for the chatbot application.

## Running Locally

1. Install dependencies:
   ```
   pip install -e .
   ```

2. Run the server:
   ```
   python main.py
   ```

The server will start at http://localhost:8000

## API Endpoints

- `/health` - Health check endpoint
- `/docs` - API documentation (Swagger UI)
